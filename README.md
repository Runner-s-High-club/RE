## runner's high club 

## 사용 스택

`Typescript` `next.js` `styled-components`

## 소개

runner's high club webView 서비스 입니다.

## 주요기능

- 웹뷰를 이용하여 서비스의 특정 기능을 분리하고 모듈화하여 유지보수성과 확장성 을 향상시켰습니다.

### Getting Started

1. `yarn` 프로젝트 의존성 설치

```
$ yarn
```

2. `dev` 프로젝트를 시작 합니다.

```
$ yarn dev
```
